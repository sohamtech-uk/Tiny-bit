# Start Here

Welcome to Tiny:bit. This guide helps you make your first robot programs.

## What Is Tiny:bit?

Tiny:bit is a small robot car for the BBC micro:bit. You can code it to move,
turn, light up, play music, follow lines, and avoid objects.

## What You Need

- A BBC micro:bit
- A Tiny:bit robot car
- A USB cable
- Batteries for the car
- A computer with Microsoft MakeCode

Open MakeCode here: https://makecode.microbit.org

## Pick Your Coding Path

If you are new, start with **Blocks** in MakeCode.

If you can read simple code, try **Python** in MakeCode.

The `Kids Projects` folder uses small Python examples because they are easy to
copy, change, and talk about.

## First 10 Minutes

1. Open MakeCode.
2. Create a new micro:bit project.
3. Add the Tinybit extension.
4. Put the micro:bit in the Tiny:bit car.
5. Try [01 Drive Forward](Kids%20Projects/01-drive-forward/README.md).
6. Change one number and see what happens.

## Three Rules For Robot Coding

1. Start slow.
2. Test one small idea at a time.
3. Change one number, then test again.
