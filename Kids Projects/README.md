# Kids Projects

These projects are short, friendly activities for learning Tiny:bit basics.

Each project has:

- a goal
- what you learn
- code to copy
- things to change
- a challenge

## Projects

1. [Drive Forward](01-drive-forward/README.md)
2. [Turn A Square](02-turn-a-square/README.md)
3. [Lights When Turning](03-lights-when-turning/README.md)
4. [Clap To Stop](04-clap-to-stop/README.md)
5. [Avoid A Wall](05-avoid-a-wall/README.md)

## Before You Start

- Put Tiny:bit on the floor.
- Start with speed `80` to `120`.
- Keep fingers away from the wheels.
- If the car goes the wrong way, stop the program and try again.
