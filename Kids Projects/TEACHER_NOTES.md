# Teacher Notes

## Suggested Lesson Flow

Time: 45 to 60 minutes

1. Show Tiny:bit and name the parts.
2. Run the Drive Forward project.
3. Ask students to change speed and pause values.
4. Try the square challenge.
5. Let students add lights, sounds, or sensors.

## Learning Goals

Students should be able to:

- explain what a motor does
- change a speed value
- use a loop
- use a condition
- test and improve a robot program

## Good Questions To Ask

- What number controls speed?
- What number controls time?
- What happens if the turn pause is too short?
- How can the robot know something is nearby?
- How can we make the robot easier to understand with lights?

## Extension Ideas

- Make a traffic-light robot.
- Build a cardboard maze.
- Add a sound when the robot stops.
- Use colored cards as checkpoints.
