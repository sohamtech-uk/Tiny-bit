# Glossary

## Code

Instructions that tell the robot what to do.

## Loop

Code that repeats. `basic.forever` repeats for as long as the micro:bit is on.

## Variable

A name that stores a value, like `speed = 100`.

## Condition

A question in code. Example: "Is the distance less than 10 cm?"

## Motor

The part that turns the wheels.

## Speed

How fast the motors move. Tiny:bit speed values often go from `0` to `255`.

## LED

A small light that can turn on, off, or change color.

## RGB

Red, green, and blue. Mixing these numbers makes different light colors.

## Sensor

A part that measures something, like sound, distance, or a line on the floor.

## Ultrasonic Sensor

A distance sensor. It uses sound to measure how far away an object is.

## Pause

A short wait in the program. `basic.pause(1000)` waits for one second.
